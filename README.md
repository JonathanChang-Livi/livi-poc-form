# livi-poc-form

> Made with create-react-library

## Install

```bash
npm install --save livi-poc-form
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'livi-poc-form'
import 'livi-poc-form/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
